Detailed Description

Thrown usually when the user inputs an invalid Duty Cycle.

This exception is usually thrown when someone attempts to set an invalid Duty Cycle on the PWM methods. The valid range is from 0 aka LOW to 255 aka 100% duty cycle always on. If outside of those ranges, this error will be thrown.
