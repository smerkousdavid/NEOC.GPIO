Handles the throwing on the return of a builtin C function.

This is one of the big ports of neo from C to C++, to actually have throwable errors and not just returned error codes. This is usually called on the end of every C function call. Yeah it has a lot of arguments, but it's easier to use this than to write it everytime.

Parameters
retTypethe returned status of the C function
pinthe pin that's being used
lowif showing low factor then put it
highhighest value if available
pddPeriod or Duty Cycle value (Usually not called within same function)
whatWhat system is being used
typeThe error message to be said if any
Note
This can accept any numerical input type
