Detailed Description

The Cpp Analog class to read pins A0 - A5.

This will read the current voltage from the analog pins on A0 through A5. Here is the example usage of the class:

int main() {
Analog azero(0);
std::cout << "Analog (A0) Read: " << azero.read() << std::endl;
return 0;
}


The m4 core must be disabled before running this method

See also
neo_disable_m4();

Analog release.

This will attempt to release the objects, if there are no more Analog Objects in use then this will attempt to run Analog::free(); aka neo_analog_free(); To stop this construct object with

Analog analog(<pin number>, false); //No release on deconstructor
Note
Do not try to mix Object oriented and static usage of any Neo Object it screws it up
When attempting to use both static Analog::read(int) and obj.read() make sure you use Analog obj(<pin>, false) to not release on 0 objects
