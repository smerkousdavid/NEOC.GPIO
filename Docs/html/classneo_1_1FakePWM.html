Detailed Description

The FakePWM class that handles all FakePWM controls (GPIO)

This class helps out getting quick and dirty PWM on any Gpio port with the Udoo Neo. I'm going to say this right now, the PWM output on the pins are not clean at all each period cycle might be different. I just have to say, it works... It's going to do It's job easily. Since you are limited on the amount of Real PWM I would recommend using FakePWM on things like LEDS where precision doesn't matter.


Here is the example usage of the class:

int main() {
FakePWM pwm(13); //Gpio pin 13
pwm.write(127); //About half duty cycle
return 0; //Auto free on exit
}
