Detailed Description

Thrown usually when the user inputs an invalid Period Nano Second.

This exception is usually thrown when someone attempts to set an invalid period on the PWM methods. Valid are 1 -> 1000000000 nano seconds. The larger the number the slower the PWM module will pulse and switch from high to low. Negative numbers are not allowed.
