Detailed Description

File to control the Builtin status LED on the board.

Author
David Smerkous
Date
11/28/2016

This is the source file that easily allows anyone to attach their program to the BuiltinLED that by default is not connected to a standard pin. This is the RED LED that indicates the file disk IO. This will be auto released on program exit, so no reason to call neo_led_free unless you want to

int neo_led_init ( )

Initializes the Builtin LED on the Udoo.

This will attach the Builtin LED to the current program

Returns
NEO_OK or NEO_UNUSABLE error if something went wrong
Warning
When using this you must be ROOT not sudo but the user root
Note
To run as root try sudo su and then run it or sudo su -c './a.out'

§ neo_led_set()

int neo_led_set ( int  state)

Sets the Builtin LED state.

This will remove the LED from the program space and return it to the system

Parameters
stateThe LED state to set it to either LOW (0) aka off or HIGH (1) aka on
Returns
NEO_OK/NEO_UNUSABLE or NEO_LED_STATE_ERROR if something went wrong
Note
Anything besides a 0 or 1 for the led state will return NEO_LED_STATE_ERROR
