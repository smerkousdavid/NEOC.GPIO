Function Documentation

§ neo_temp_free()

int neo_temp_free ( )

Frees the controller from the current program.

This will release the temperature brick controller from the program

Returns
NEO_OK or NEO_UNUSABLE_ERRROR if it failed to release the brick

§ neo_temp_init()

int neo_temp_init ( )

Initializes the temperature controller.

Starts the driver for the temperature brick module to be able to read data from it. Extremely simple use and conversion functions.

Returns
NEO_OK or NEO_UNUSABLE_ERRROR if it failed to load the temperature driver or find the brick

§ neo_temp_read()

int neo_temp_read ( )

Reads the current temperature from the brick module.

Read the current temperature from the brick snapin sensors. The raw return of this function is in milli celcius.

Returns
NEO_OK or /NEO_READ_ERROR/NEO_UNUSABLE_ERRROR if it failed to read from the brick
