Sets millis pulling rate of magno. 

This will pull the new magnometer values every X millis The faster the poll rate the more C\+PU will be used to process the values Also the more distant the values will be. It\textquotesingle{}s recommended to only up this value If you don\textquotesingle{}t set the poll rate fast enough you might get the same value twice when using the read method
